# TAPWM - Técnicas Avançadas de Programação Web e Mobile

Repositório destinado às atividades e projetos avaliativos da disciplina de **Técnicas Avançadas de Programação Web e Mobile**.

## 📌 Sobre o Repositório
Este espaço consolida as entregas e os desafios práticos desenvolvidos ao longo das aulas. O foco é a construção de aplicações web funcionais, manipulação do DOM e a aplicação de boas práticas em interfaces de usuário, utilizando essencialmente as tecnologias fundamentais (e a transição de outras linguagens para web).

---

## 📁 Atividades Concluídas

### 🖥️ [Atividade 2 - Windows Forms App](./Atividade2/)
Estrutura inicial de um projeto (em pasta de projeto C# Windows Forms). Abordagem de componentes visuais tradicionais focados em aplicativos de janela (Desktop).

### 🧮 [Atividade 3 - Média e Operações Básicas em JS](./Atividade3/)
Páginas simples introdutórias ao uso de variáveis e interações pontuais por blocos de diálogos no JavaScript.
- **Media:** Solicita o nome e o registro de notas escolares de um aluno, aplicando a matemática para dar o boletim ou resultado final da média.
- **Operacoes:** Utilizado para capturar informações de valores literais do cliente e simular processamento e devolução de contas matemáticas de base.

### ✂️ [Atividade 4 - Jogo Pedra, Papel ou Tesoura](./Atividade4/)
Mini-game web (conhecido como *Jo-Ken-Po*) totalmente interativo.
- Dispõe de interface bonita e moderna construída com CSS3 flexível.
- Script faz o rastreamento das escolhas do usuário de um lado, e gera respostas usando cálculos de matriz e aleatoriedade (`Math.random()`) simulando a máquina do outro.
- Verifica de forma imediata e diz se a rodada deu vitória, derrota ou empate na tela para o jogador.

### 🎬 [Atividade 5 - Pesquisa de Opinião sobre o Filme](./Atividade5/)
Aplicação front-end simples para consolidar lógica de manipulação de dados em array dinâmicos e injeção do DOM.
- Recebe de cadastros preenchidos manualmente pelo usuário: Idade, Sexo e Nota até bater sua proteção de limite de 45 registros máximo.
- Imprime as médias instantâneas, encontra o participante listado mais velho e também o mais novo. Destaca com totalizadores a aprovação visual dessa demografia.
- Trabalha somente em Memória Volátil de tempo de execução, apagando após qualquer reload (F5).

---

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Utilizado para base e a elaboração estrutural, além das tags de inputs visuais.
- **CSS3:** Interfaces de design sem dependência de dependências, uso de cores e Grid.
- **JavaScript (Vanilla JS):** O motor inteiro de cálculos de todas as atividades, manipulação das abas e condicionais (`if`, `for`).

## 🚀 Como Executar

Por grande parte das pastas abordarem apenas elementos nativos (Front-End web), não é demandada a instalação de interpretadores nos bastidores para ver o código em ação.

1. Baixe os recursos diretamente desta mesma página (seja como Zip) ou rodando comando via seu controle de versão Git:
   ```bash
   git clone https://github.com/seu-login/TAPWM.git
   ```
2. Após salvar em seu HD, apenas caminhe entre os diretórios e selecione a tarefa desejada.
3. Arraste ou abra normalmente qualquer documento do tipo `.html` (ex: `index.html` ou `Media.html`) no seu Browser padrão (Google Chrome, Edge, etc.)
